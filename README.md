Datamaps
=====

### Interactive maps for data visualizations. Bundled into a single Javascript file.


The goal of Datamaps is to help create geographical based data visualizations. Currently it supports choropleths and bubbles. 

Read more (and demos) at http://datamaps.github.io

Downloads:

 - [World map](http://datamaps.github.io/scripts/datamaps.world.min.js)
 - [USA only](http://datamaps.github.io/scripts/datamaps.usa.min.js)
 - [USA & World](http://datamaps.github.io/scripts/datamaps.all.min.js)
 - [No preset topojson](http://datamaps.github.io/scripts/datamaps.none.min.js)

Examples:

  - [Datamap of ongoing military conflicts](http://bl.ocks.org/4127667)
  - [US Election map](http://bl.ocks.org/4127517)
